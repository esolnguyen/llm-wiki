# Concepts

Top-level taxonomy of concept pages, grouped by topic. Each subdirectory has its own `index.md` listing the pages it contains.

## Topics

- [[productivity/index|Productivity]] — time management, focus, attention (1 concept, 1 source)

## Hierarchy rules

- A new top-level topic is added when 2–3 concepts cluster outside any existing branch.
- A sublevel (e.g. `productivity/time-management/`) is added when a topic accumulates ~8+ concepts that cluster naturally.
- A concept lives in **one** primary folder. Cross-topic relationships are expressed via wikilinks, not duplication.
- Each folder's `index.md` lists its child concepts and any sub-topics, with one-line summaries.
